What is "BataGacha!"

Shut doors forever in BataGacha!. The rules are simple. Just press buttons quickly and accurately depending on whatever door comes your way. The controls are easy but addictive?once you start locking doors, you can't stop! See the tutorial and in-game manual for more information on the controls.

  • An in-game manual with a tutorial and an explanation of the rules has been added.
  • A good door effect has been implemented.
  • Some layout settings have been removed.
  • The option to display Jill and Graham in front of the doors for the ST Edition has been added.

  • The option to change the app icon to the old version of the icon has been added.
  • An English translation has been implemented.

  • The score calculation algorithm has been changed.
  • Banner ads have been removed.
